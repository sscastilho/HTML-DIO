História dos computadores

(Voltar)

Primeiros Computadores: A história dos computadores são marcadas por gerações, sendo um marco na primeira geração: Harvard Mark 1 (1944), Colossus (1946) e ENIAC (1946). Na segunda geração (1959 a 1965), os computadores se destacam pelo uso de Transistores. Na terceira geração (1965 a 1970), os computadores eram bem menores e em sua estrutura eletrônica era composta de circuito integrado, transistor, capacitores e resistores. Na quarta geração, a partir de 1971 os computadores diminuiram de tamanho e surgiram os microprocessadores, a memória endereçada aumentou para 64KB. Na quinta geração (atualmente), as principais tecnologias aplicadas são Inteligência artificial, multiprocessadores e nanotecnologia. Os exemplos de dispositivos que fazem uso das novas tecnologias são: Laptops e Smartphones.

O que são Clients?

(Voltar)

Cliente: é uma definição de nome em computação e representa uma entidade que consome os serviços de uma outra entidade servidora, por meio do uso de uma rede de computadores em uma estrutura de cliente-servidor.

O que são Servers?

(Voltar)

Servers: é um software ou pc mais potente que um computador comum com sistema centralizado que fornece serviços a uma rede de computadores chamada de "cliente".

Como é uma estrutura básica de um HTML?

(Voltar)

HTML: O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Segue abaixo a estrutura de como ela inicia:

<!DOCTYPE html>
<html>
<head>
<title>Como exibir código HTML
<meta charset="utf-8">
</head>
<body>
<p>Eu quero exibir a tag <h1> aqui!</p>
</body>
</html>

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Listas ordenadas e não ordenadas: A diferença entre as listas ordenadas e não-ordenadas está na maneira com que os itens de cada lista são apresentados por exemplo: a lista ordenada enumera os elementos já a lista não-ordenada coloca os marcadores antes de cada elemento.
